# Disaster Response Rover

A smart search-and-rescue rover designed to assist emergency response teams in locating trapped victims in disaster-affected areas. The system combines machine learning, IoT sensors, live video streaming, and Android-based remote control to improve safety and efficiency during rescue operations.

## Overview

Natural disasters such as earthquakes, landslides, and building collapses often create environments that are dangerous and difficult for rescuers to access. This project introduces a remotely operated rover capable of navigating hazardous terrain, detecting human presence, monitoring environmental conditions, and enabling communication with victims.

## Features

- Human detection using Machine Learning
- Live video streaming through Raspberry Pi Camera
- Android application for remote control and monitoring
- Obstacle detection using Ultrasonic Sensors
- Human presence sensing using IR Sensors
- Temperature and humidity monitoring
- Speech-to-Text communication
- Text-to-Speech communication
- Tank-track mobility for rough terrain navigation
